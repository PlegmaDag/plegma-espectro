O serviço pode ser cancelado a qualquer momento via Dashboard. Não há reembolso proporcional ao período restante do ciclo anual em curso.

O Lattice Shield é uma camada adicional de proteção e não substitui boas práticas de segurança. A rede PLEGMA DAG não se responsabiliza por danos decorrentes de ameaças anteriores à instalação do agente ou de ameaças não contempladas na base de assinaturas vigente.

@contract AcessoStaking {

  @state
  map<PLGAddress, uint256> stake_saldo;
  map<PLGAddress, uint256> stake_tempo;
  uint256 minimo_acesso = 1000;

  @entry depositar(uint256 valor) {
    require(valor > 0, "Valor inválido");
    stake_saldo[caller] += valor;
    stake_tempo[caller] = block.timestamp;
  }

  @view tem_acesso(PLGAddress addr) → bool {
    return stake_saldo[addr] >= minimo_acesso;
  }

  @entry sacar(uint256 valor) {
    uint256 lock = stake_tempo[caller] + 30 days;
    require(block.timestamp > lock, "Lock 30d ativo");
    stake_saldo[caller] -= valor;
  }
}
