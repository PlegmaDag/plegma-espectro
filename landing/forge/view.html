Como usar em qualquer site

1. Inclua o player
<script src="zkm-player.js"></script>
2. Use como tag HTML nativa
<!-- Tag Web Component — funciona como <img> -->
<zkm-img src="minha-imagem.zkm" width="400"></zkm-img>
3. Ou via JavaScript
// Renderiza em qualquer elemento pelo seletor CSS
ZKM.render('minha-imagem.zkm', '#meu-container');
4. O que acontece automaticamente
⬡ Prova SHA-256 verificada no carregamento — exibe ZK-VERIFIED ✓ ou PROVA INVÁLIDA ✗
⬡ Animação de montagem no primeiro carregamento (pixels aparecem gradualmente)
⬡ GIF animado reproduz com timing original + efeito visual ZKM
⬡ Shadow DOM isolado — sem conflito com CSS do site hospedeiro
