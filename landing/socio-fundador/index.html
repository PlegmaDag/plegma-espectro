Guia — Configurar Nó Minerador em VPS
Azure — Trial Gratuito 30 Dias (Microsoft)
⚠ O trial Azure inclui $200 USD em crédito por 30 dias. Não é cobrado automaticamente se você não upgradar para pay-as-you-go após o período.

Passo 1 — Acesse azure.microsoft.com/free e crie a conta com email Microsoft. Exige cartão de crédito para verificação (não cobrado no trial).

Passo 2 — No portal Azure, crie uma nova Virtual Machine:

  • Região: East US ou West Europe (menor latência)
  • Sistema: Ubuntu Server 22.04 LTS
  • Tamanho: B2s (2 vCPU · 4 GB RAM) — suficiente para nó Validator
  • Autenticação: SSH public key (gere um par com ssh-keygen)
  • Portas abertas: 22 (SSH), 8080, 8082, 8083

Passo 3 — Após criar a VM, copie o IP público. Conecte: ssh -i ~/.ssh/id_rsa azureuser@IP_DA_VM

Passo 4 — Execute o script de instalação do nó (ver painel ao lado).

O trial B2s consome ~$0,04/h. Com $200 de crédito você tem ~208 dias. Após o trial, B2s fica ~$29/mês.

Script de Instalação do Nó PLEGMA 
#!/bin/bash
# =============================================================
# PLEGMA DAG — Script de Instalação do Nó (Ubuntu 20.04+)
# Execução: chmod +x install_node.sh && sudo ./install_node.sh
# =============================================================

set -e
echo "==================================================="
echo " [*] PLEGMA DAG — Instalação do Nó Minerador"
echo "==================================================="

# Atualizar sistema
apt update && apt upgrade -y

# Dependências base
apt install -y python3 python3-pip python3-venv git curl wget unzip \
               build-essential libssl-dev libffi-dev openssl

# Python 3.10+ necessário para dilithium-py
python3 --version

# Clonar ou baixar PLEGMA_CORE
mkdir -p /opt/plegma
cd /opt/plegma

# Se você tem o zip do PLEGMA_CORE:
# wget https://plegmadag.com/download/plegma_core.zip && unzip plegma_core.zip

# Virtualenv
python3 -m venv venv
source venv/bin/activate

# Dependências Python
pip install --upgrade pip
pip install blake3 py_ecc dilithium-py flask requests

# Configurar firewall (UFW)
ufw allow 22/tcp     # SSH
ufw allow 8080/tcp   # API DAG
ufw allow 8082/tcp   # Auth
ufw allow 8083/tcp   # Wallet
ufw --force enable

# Criar serviço systemd para core_vm.py
cat > /etc/systemd/system/plegma-node.service <<EOF
[Unit]
Description=PLEGMA DAG Node
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/plegma/PLEGMA_CORE
Environment=PATH=/opt/plegma/venv/bin
ExecStart=/opt/plegma/venv/bin/python3 core_vm.py
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
EOF

systemctl daemon-reload
systemctl enable plegma-node
systemctl start plegma-node

echo ""
echo "==================================================="
echo " [✓] Nó PLEGMA iniciado!"
echo " [*] Verificar status: systemctl status plegma-node"
echo " [*] Ver logs: journalctl -u plegma-node -f"
echo " [*] API acessível em: http://$(curl -s ifconfig.me):8080/api/status"
echo "==================================================="
